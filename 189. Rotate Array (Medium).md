## **189. Rotate Array (Medium)** 

https://leetcode.com/problems/rotate-array/



#### Ideas : 

1.  Solution 1 is based on reverse. Once we reverse the while vector, it will be easier to reach the answer.
2.  Solution 2 is based on swap. There are two cases : left side is larger, and right side is larger. Every swap might change the case. Keep swapping them until k is less than 0.



#### Solution 1

Language : C++

```C++
class Solution 
{
public:
    void rotate(vector<int>& nums, int k) 
    {
        k %= nums.size();
        reverse(nums.begin(), nums.end());
        reverse(nums.begin(), nums.begin()+k);
        reverse(nums.begin()+k, nums.end());
    }
};
```



#### Solution 2

Language: C++

```c++
class Solution 
{
public:
    void rotate(vector<int>& nums, int k) 
    {
        int start = 0, tmp = 0, n = nums.size();
        k %= n;
               
        while(k > 0)
        {
            if(n - k >= k)
            {
                for (int i = 0; i < k; i++)
                    swap(nums[start + n - 2*k + i], nums[start + n - k + i]);
                
                n = n - k;
                k = k%n;
            }
            else
            {
                for (int i = 0; i < n - k; i++)
                    swap(nums[start + i], nums[start + n - k + i]);

                tmp = n - k;
                n = k;
                k -= tmp;
                start += tmp;
            }
        }
    }
};
```
